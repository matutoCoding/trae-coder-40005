## 1. 产品概述

粉末冶金铁基零件生产管理系统是一款面向粉末冶金制造企业的客户端软件，用于管理从粉末配料到成品检验的完整生产流程。系统覆盖粉末配料、混料制粒、压制成型、高温烧结、复压整形、浸油处理、尺寸检验七大核心工序模块，帮助企业实现生产数据数字化、过程可追溯、质量可控的目标。

- **目标用户**：粉末冶金厂生产管理人员、工艺工程师、质量检验员、车间操作员
- **核心价值**：实现生产全过程数据记录与追溯，提升产品质量稳定性，优化生产效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 账号密码 | 全部功能管理、用户管理、数据配置 |
| 工艺工程师 | 账号密码 | 工艺参数配置、生产数据查看与分析 |
| 操作员 | 账号密码 | 对应工序数据录入、生产记录查看 |
| 质检员 | 账号密码 | 检验数据录入、质量报告查看 |

### 2.2 功能模块

1. **粉末配料模块**：金属粉末配比管理、配料记录、配方管理
2. **混料制粒模块**：润滑剂混料、混料时间、制粒参数记录
3. **压制成型模块**：压坯密度控制、压制压力记录、成型参数
4. **高温烧结模块**：烧结炉温曲线、保护气氛监控、烧结炉次台账
5. **复压整形模块**：复压整形量记录、整形尺寸控制
6. **浸油处理模块**：含油轴承浸油、浸油时间、含油率检测
7. **尺寸检验模块**：零件密度测量、尺寸公差检验、裂纹外观检查

## 3. 核心流程

生产流程按照粉末冶金工艺顺序进行：粉末配料 → 混料制粒 → 压制成型 → 高温烧结 → 复压整形 → 浸油处理 → 尺寸检验。每个工序环节的数据与批次号关联，形成完整的产品追溯链。

```mermaid
flowchart LR
    A["粉末配料"] --> B["混料制粒"]
    B --> C["压制成型"]
    C --> D["高温烧结"]
    D --> E["复压整形"]
    E --> F["浸油处理"]
    F --> G["尺寸检验"]
    G --> H["成品入库"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：工业蓝（#165DFF）作为主色，体现制造业专业稳重特质
- **辅助色**：钢灰色（#4E5969）、警示橙（#FF7D00）、成功绿（#00B42A）
- **中性色**：深灰（#1D2129）、中灰（#4E5969）、浅灰（#C9CDD4）、背景灰（#F2F3F5）
- **按钮风格**：直角微圆角（4px）、实心按钮主色填充、次要按钮描边
- **字体**：使用系统无衬线字体，标题加粗，正文常规
- **布局风格**：左侧导航栏 + 顶部标题栏 + 主内容区，卡片式布局
- **图标风格**：线性图标，24px标准尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 仪表盘 | 数据概览 | 统计卡片网格、趋势图、炉次列表 |
| 工序页面 | 各模块 | 左侧表单区、右侧数据表格、状态标签 |
| 烧结页面 | 高温烧结 | 温度曲线图、气氛监控面板、炉次台账表 |
| 检验页面 | 尺寸检验 | 测量数据表单、公差判定、检验结果汇总 |

### 4.3 响应式
- 桌面优先设计，最低支持 1366px 宽度
- 侧边栏可折叠，适配不同屏幕尺寸
- 表格区域支持横向滚动

### 4.4 动效设计
- 页面切换采用淡入淡出过渡
- 数据加载使用骨架屏占位
- 按钮、卡片悬停状态有微交互
- 图表数据加载时使用渐显动画
